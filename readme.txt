This plugin allows you to add Monegraph tooltips to your WordPress blog.

== Description ==

Monegraph Everywhere allows anyone to easily link Monegraph work numbers, or "mURLs," back to the registered work on Monegraph.  Any text matching a mURL will be automatically hyperlinked and will show a summary tooltip of the work on mouse hover-over.

== Installation ==

To install the Monegraph Everywhere plugin:

1. Install the plugin through the WordPress plugins screen, or upload the plugin files to the `/wp-content/plugins/monegraph-everywhere` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. (Optional) If you have an Affiliate ID, use the Settings->Monegraph Everywhere screen to enter it.
1. You are now up and running!  Monegraph work numbers, or "mURLs", will now be detected within any page on your blog.

== Frequently Asked Questions ==

= Do I need a Monegraph account to use this plugin? =

A Monegraph account is not required to use this plugin.
